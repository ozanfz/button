# Button
# Cara Mengakses fitur run command:
Tekan Ctrl + R/R (Windows/Linux) atau Cmd + R (Mac)
Dialog Run akan muncul dengan gaya Windows
Perintah yang Tersedia:
📋 Aksi Utama:
reset
galeri / gallery - Buka galeri latar belakang
🌓 Mode:
dark / gelap - Ganti ke dark mode
light / terang - Ganti ke light mode
🎨 Bentuk Tombol:
kotak / square - Ubah tombol jadi kotak
bulat / rounded - Ubah tombol jadi rounded
lingkaran / circle - Ubah tombol jadi lingkaran
👆 Interaksi:
klik / click - Otomatis klik tombol
